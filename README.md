Reparto sugerido y estructura de niveles

Pequeña página estática para tener a mano los repartos de fichas por jugador y la estructura de niveles de ciegas.

Archivos:
- index.html — Interfaz principal
- styles.css — Estilos
- script.js — Datos y pequeñas utilidades (copiar/imprimir/vista compacta)

Novedades:
- Añadida sección de inventario con cálculo total de fichas y puntos.
- Diseño optimizado para móvil (tablas responsivas y botones accesibles).

Cómo usar:
- Abre `index.html` en un navegador (doble clic o desde Live Server)
- Usa "Copiar datos" para copiar todo al portapapeles
- Usa "Imprimir" para sacar una versión física
- Pulsa "Vista compacta" para reducir el espaciado

Puedes editar los valores dentro de `script.js` en la constante `data` si quieres ajustar stacks o niveles.
